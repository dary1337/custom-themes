# How to install ?

-    Download [zip](https://github.com/dary1337/custom-themes/archive/refs/heads/master.zip)
-    Unpack
-    Go to `chrome://extensions/` tab
-    Enable developer mode
-    Click "Load unpacked"
-    Choose unpacked folder
-    Enable extension in incognito (if needed)
-    Done
